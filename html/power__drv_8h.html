Detailed Description

Atmel file. Power Management low level driver definition.

Copyright (c) 2006 Atmel.

Use of this program is subject to Atmel's End User License Agreement. Please read file license.txt for copyright notice.

This module allows to configure the different power mode of the AVR core and also to setup the the internal clock prescaler

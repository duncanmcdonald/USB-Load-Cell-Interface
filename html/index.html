USB interface

This device implements End Point 0 for USB commands and End Point 1 for Interrupt in

  • USB commands are initiated by the host for EP 0
  • TEDS data is also sent via a HID record in EP 0
  • Measured data is sent in HID records via device-to-host initiated query on EP1

firmware structure

Has three components:

  1. Main loop
    • Initialization
    • Poll USB looking for device requests
  2. Timer interrupt service that controls ADC sampling rate
  3. SPI bus interrupt service when ADC has completed a measurement

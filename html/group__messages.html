Detailed Description

Defines structures for all inbound and outbound messages.

It is intended to be included by the device and host programs wishing to exchange messages with the device. It uses fundamental data types: U8 U16 U32

DEVICE_STATE_ACQUIRING.

When the device is actively acquiring data, the only allowable operation is to stop acquiring. In this state, the device is sending IN reports to the host at a constant interval.

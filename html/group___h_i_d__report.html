Defines for reports.

Request pages of TEDS data from the device (if the TEDS_STATE is such that TEDS data has been read). Each page is 32 bytes long and is addressed with a different report id (0, 1, 2, 3, 4). Each of these returns a REPORT_FEATURE_QUERYTEDS_PAGE_X structure

void appusbsensor_poweron_stable ( )

Definition at line 331 of file appusbsensormain.c.

U32 global_state_poweronticks

If this value is >0, then it will be decremented each time through the main loop. When it reaches zero, the variable global_state_poweron_stable is set to 1, which indicates that the requisite amount of time has passed after usb configuration and "dangerous" things can be done safely.

#define REPORT_OUT_CMD_RESET_TEDS   30

Command to reset/re-read TEDS data.

This command can only be issued from the IDLE device state. It results in the device temporarily transitioning to the DEVICE_STATE_READINGTEDS state. The host should query the teds state after sending this command until the TEDS state is TEDS_STATE_READY.
